vimrc読書会まとめ

https://vim-jp.org/reading-vimrc/


:memo: 更新手順
---------------

Gitter 上の bot で更新できます。

1. 終了時(`!reading_vimrc finish`)の更新は自動で行われます。Wiki も自動で更新されます。
2. 以下のコマンドで次回分を設定します。

```
!reading_vimrc next {next_vimrc_github_urls...}
```

3. 完了 :tada:


:book: Wiki更新手順
-----------------------------

bot からリクエストを追加できます。

```
!reading_vimrc request {vimrc_github_url} [comment]
```
